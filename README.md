# recast-ast

## Project setup
```
git clone https://github.com/zwf193071/recast-ast.git
```
Then in your terminal, you can type as following

```
exportsFn -h
```
Or

```
exportsFn --help
```
you can see the helpful information

### How to use
```
exportsFn demo.js
```
You can see the export.demo.js in the current directory
